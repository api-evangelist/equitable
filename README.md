# Equitable (equitable)

Equitable Holdings, Inc. is a financial services holding company offering individual retirement, group retirement, life insurance, and employee benefits solutions. In 2025, Equitable launched new API capabilities for benefits plan setup integration.

## APIs

### Equitable Benefits Plan Setup API

An API capability that acts as a digital bridge between systems, enabling automatic creation and configuration of employee benefits plans for small and medium-sized businesses. Equitable has partnered with Employee Navigator to add API functionality that helps brokers build groups and products directly in their platform, allowing them to move from sale notification to enrollment readiness in hours.

**Note:** This API is available to brokers through the Equitable digital onboarding process. No public developer portal or self-service API documentation has been identified.
