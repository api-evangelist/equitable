## APIs

### Equitable Benefits Plan Setup API

An API capability that acts as a digital bridge between systems, enabling automatic creation and configuration of employee benefits plans for small and medium-sized businesses. Integrated with Employee Navigator to streamline group and product setup for brokers, allowing them to move from sale notification to enrollment readiness in hours.

**Note:** This API is available to brokers through the Equitable digital onboarding process. No public developer portal or self-service API documentation has been identified.
